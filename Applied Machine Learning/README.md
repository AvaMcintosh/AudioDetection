### Web-design to Custmize earbuds to your comfort

This project is a web-based customization platform that allows users to design their own wireless earbuds by selecting colors, patterns, finishes, and design themes.

The goal is to explore personalized consumer hardware design, while connecting future work in ear scanning, comfort optimization, and audio intelligence.

Users can design their own earbuds visually, see changes update live and make it personalizes before purchase.

# Problem
Most earbuds are:
- One-size-fits-all
- Limited in design options
- Not personalized for comfort or identity

This project explores how customization, user choice, and future ear-scanning data could improve comfort, ownership, and user experience.
